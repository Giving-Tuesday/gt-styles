Secondary typeface for body copy and extended reading.

Brand Rules

Design Guidelines

Important rules to maintain brand consistency.

Forbidden Color Combinations

These color combinations should NEVER be used together:

+
Never combine Red + Lavender

This combination creates visual conflict and should be avoided in all materials.

Typography Restrictions

The following colors should NOT be used for text:

  • Light Blue (#6CF4FF) - Too bright for readable text
  • Lavender (#CAC7F8) - Insufficient contrast for body text
  • Beige (#FDF7F7) - Only suitable on dark backgrounds

Contrast Requirements

All text must meet WCAG 2.1 AA contrast requirements:

  • Normal text: 4.5:1 minimum contrast ratio
  • Large text (24px+): 3:1 minimum contrast ratio
  • Use Navy (#001548) or Black (#000000) for body text on light backgrounds
  • Use White (#FFFFFF) or Beige (#FDF7F7) for text on dark backgrounds
